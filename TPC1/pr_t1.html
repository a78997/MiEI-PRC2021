Abstract:

Neste trabalho prático o objetivo era gerar um dataset em Turtle, de acordo com os requisitos da ontologia desenvolvida na aula. Os requisitos eram a criação de um dataset composto por 4 UC's, 4 docentes e 200 alunos, tendo em conta os relacionamentos previamente definidos.

Para isto foi utilizado um gerador de JSON para inicialmente criar o dataset em JSON com todas as entidades pretendidas.

O passo seguinte foi desenvolver um notebook Python no JupyterLab e, através da biblioteca pandas, ler o dataset JSON e converter os dados para Turtle.

Finalizada a conversão o último passo foi carregar o dataset desenvolvido para o Protege, de forma a confirmar que as instâncias e os relacionamentos tinham sido bem desenvolvidos.
