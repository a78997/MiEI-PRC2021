Abstract:

Neste trabalho foi pedido que fosse desenvolvida uma ontologia no Protégé de forma a representar toda a informação disponível no dataset do arquivo musical fornecido.

De forma a povoar esta ontologia, foi necessário desenvolver um conversor em XSL de XML para TTL de forma a povoar a ontologia com a informação do dataset.

Os ficheiros utilizados para a realização deste trabalho estão disponíveis em baixo, desde a Ontologia desenvolvida, a stylesheet de conversão e a ontologia final já povoada.

Deliverables:

  • Página HTML - Manifesto
  • Ficheiro DTD - Manifesto
  • Stylesheet - Manifesto
  • Documento XML - Manifesto
  • Arquivo Musical - dataset XML
  • Ontologia Inicial - classes, atributos e relacionamentos
  • Ontologia Povoada
  • Conversor XSL de XML para TTL
